ESP32 C6 Dev Board

Below are the additional board URL:
1. https://espressif.github.io/arduino-esp32/package_esp32_dev_index.json
2. https://espressif.github.io/arduino-esp32/package_esp32_index.json

For the first time:

1. Go to Select other boards and port
2. Under board select ESP32C6 Dev Module
3. Under ports select the port where you have connected the ESP32C6
4. Click Ok
5. After that you can write your program (or use one of the existing examples) and upload it to the ESP32C6 Dev Board.
